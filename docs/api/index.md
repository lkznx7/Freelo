# API Documentation

A documentação da API do sistema Freelo (Escala Fácil) segue o padrão OpenAPI 3.1.

## Especificação Técnica
O contrato completo da API pode ser encontrado em:
[OpenAPI Specification (YAML)](../openapi/spec.yaml)

## Endpoints Principais
- **/api/auth**: Gestão de autenticação e tokens.
- **/api/users**: Perfis de Workers e Companies.
- **/api/jobs**: Publicação e busca de vagas.
- **/api/applications**: Candidaturas e processos seletivos.
- **/api/wallet**: Transações e saldos PIX.
- **/api/events**: Gestão operacional de eventos.

Para visualizar a documentação de forma interativa, você pode importar o arquivo YAML em:
