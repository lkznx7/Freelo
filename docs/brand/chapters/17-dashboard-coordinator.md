# Capítulo 17: Dashboard Coordinator

## Foco: Operação e Velocidade
Uma interface otimizada para uso em campo (mobile-first).

### Elementos Principais:
- **Lista de Presença:** Filtros rápidos por "Presentes", "Atrasados" e "Faltantes".
- **Check-in via Scanner:** Botão flutuante para leitura de QR Code/ID.
- **Alertas Urgentes:** Notificações de substituição necessária.
