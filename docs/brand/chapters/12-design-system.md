# Capítulo 12: Design System

## Filosofia
O Design System da Freelo é baseado em **Atomics**. Cada componente é construído para ser reutilizável e consistente em todos os 8 módulos.

## Design Tokens
- **Radii:** `2xl` (16px), `3xl` (24px), `full` (9999px).
- **Spacings:** Sistema de 4px (Base 4). `p-4` (16px), `p-6` (24px).
- **Typography:** `text-xs` (12px), `text-sm` (14px), `text-base` (16px), `text-5xl` (48px).
