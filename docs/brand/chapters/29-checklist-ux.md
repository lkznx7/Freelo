# Capítulo 29: Checklist de UX

- [ ] O usuário consegue entender o status da sua tarefa em menos de 2 segundos?
- [ ] O caminho para a ação principal (CTA) está desobstruído?
- [ ] Existem feedbacks visuais (toasts/loaders) para todas as ações assíncronas?
- [ ] A navegação mobile permite uso com apenas uma mão (botões ao alcance do polegar)?
- [ ] As mensagens de erro são instrutivas e não apenas punitivas?
