# Capítulo 19: Arquitetura do Produto

## Frontend: Next.js + Tailwind CSS
- **App Router:** Navegação moderna e otimizada.
- **Client Components:** Para interatividade rica nos dashboards.
- **Server Components:** Para SEO e performance nas páginas públicas.

## Backend: Spring Boot + Spring Modulith
- **Modular Monolith:** Garantia de separação de domínios sem a complexidade prematura de microsserviços.
- **Domain-Driven Design (DDD):** Arquitetura guiada pelo negócio.
