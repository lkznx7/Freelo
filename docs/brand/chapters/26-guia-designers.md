# Capítulo 26: Guia para Designers

## Grid e Alinhamento
- Utilize o grid de 12 colunas para desktop.
- Mantenha espaçamentos consistentes de 24px (p-6) em containers principais.

## Ícones
- Utilize exclusivamente a biblioteca **Lucide React**.
- Stroke width sugerido: 2px.

## Micro-interações
- Todas as ações de hover devem ter uma transição suave (duration-300).
- Botões devem possuir estado "active" com escala de 0.95.
