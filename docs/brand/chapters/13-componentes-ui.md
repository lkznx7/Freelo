# Capítulo 13: Componentes UI

## 1. Cards (Container de Informação)
- Bordas arredondadas (32px).
- Bordas sutis (border-border).
- Hover effect: Shadow suave e leve elevação.

## 2. Botões (Actions)
- **Primary:** Laranja com sombra laranja suave.
- **Secondary:** Azul marinho.
- **Ghost/Outline:** Para ações secundárias.

## 3. Badges (Status)
- **Padrão:** Fundo colorido com opacidade baixa (10-20%) e texto na cor sólida.
- **Formato:** Rounded-xl para manter a linguagem moderna.
