# Capítulo 7: Identidade Visual

## Conceito Visual
A identidade visual da Freelo é construída sobre o pilar da **"Energia Profissional"**. Combinamos a vibração do laranja com a sobriedade do azul marinho para transmitir um ambiente de oportunidade séria.

## Estilo de Design
- **Minimalismo Funcional:** Espaços em branco generosos para facilitar a leitura.
- **Cantos Arredondados (32px / 40px):** Transmite modernidade e acessibilidade ("Software Moderno").
- **Glassmorphism:** Uso sutil de transparências e borrões (backdrops) para profundidade.
- **Sombra Suave (Soft Shadows):** Elevação de componentes para hierarquia clara.
